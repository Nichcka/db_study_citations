# Citations DB

Веб-приложение для работы с базой данных таксонов и научных статей. Позволяет просматривать, фильтровать, добавлять и удалять записи через браузер.

---

## База данных

Три таблицы SQLite:

**Article** — научные статьи:

| Поле | Тип | Описание |
|------|-----|----------|
| pmid | INTEGER PK | PubMed ID |
| pmc_id | TEXT | PubMed Central ID |
| doi | TEXT | DOI статьи |
| title | TEXT | Название статьи |
| pub_year | INTEGER | Год публикации |
| authors | TEXT | Авторы |
| fetched_at | TEXT | Дата добавления (YYYY-MM-DD) |

**Taxon** — таксоны из NCBI Taxonomy:

| Поле | Тип | Описание |
|------|-----|----------|
| tax_id | INTEGER PK | NCBI Taxonomy ID |
| name | TEXT | Название таксона |
| rank | TEXT | Ранг (genus, species и т.д.) |
| parent_tax_id | INTEGER | ID родительского таксона |

**Taxon_article** — связь таксонов со статьями (many-to-many):

| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER PK | Суррогатный ключ |
| tax_id | INTEGER FK | Ссылка на Taxon |
| pmid | INTEGER FK | Ссылка на Article |

# Установка и запуск

## Установить зависимости
```bash
uv sync
```

## Запустить сервер
```bash
uv run uvicorn main:app
```

Открыть в браузере: http://127.0.0.1:8000

## Возможности интерфейса

### Таблица

На главной странице отображается сводная таблица. Каждая строка содержит данные и о таксоне, и о связанной статье.

**Колонки:** Tax ID, Taxon name, Rank, Parent tax, PMID, PMC ID, DOI, Title, Year, Authors, Fetched at.

### Фильтрация

- Клик на иконку 🔍 в заголовке любого столбца открывает поле поиска по этому столбцу
- Поиск регистронезависимый, по подстроке
- Несколько фильтров работают одновременно
- В левой панели: сортировка по `fetched_at` (↑↓) и чекбокс «Только с PMC ID»
- Кнопка «Сбросить фильтры»

### Выбор колонок

В левой панели галочками выбираются видимые столбцы. 

### Пагинация

100 строк на страницу. Навигация по страницам сохраняет все активные фильтры.

### Добавление записей

Кнопка «Добавить таксон» открывает форму, которая за одно действие:
1. Создаёт запись в таблице `Taxon`
2. Создаёт запись в таблице `Article`
3. Автоматически создаёт связь в `Taxon_article`

Обязательные поля отмечены `*`.

### Удаление

Кнопка 🗑 в каждой строке удаляет связь из `Taxon_article` с подтверждением («Вы уверены?»).
